# react-minesweeper

A SPA version of Minesweeper.

## Demo

https://protected-sierra-52671.herokuapp.com/

### Installing

Install dependencies with npm

```
npm install 
```

### Run on webpack dev server

```
npm run start
```

The app will be running on localhost:8080
